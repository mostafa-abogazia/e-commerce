Create a complete, modern, fully responsive E-Commerce Website using **React.js**.

The website must fetch real products from:
**https://fakestoreapi.com/products**

--------------------------------------------
📌 Technologies:
--------------------------------------------
• React.js  
• React Router  
• Fake Store API  
• Context API for Cart  
• CSS Modules or TailwindCSS (choose the best visually)  

--------------------------------------------
📌 Pages to generate:
--------------------------------------------
1) Home Page  
2) Products Page (fetch from Fake Store API)  
3) Product Details Page (fetch dynamically by ID)  
4) Cart Page  
5) Checkout Page  
6) Wishlist Page  
7) About Page  
8) Contact Page  
9) Search Page  
10) Login / Register Page  

--------------------------------------------
📌 Components needed:
--------------------------------------------
• Responsive Navbar with logo, links, cart icon (with counter), search bar  
• Footer with links  
• Product Card: image, title, category, price, rating, button  
• Category buttons  
• Hero Section  
• Filter Section  
• Cart Item component  
• Forms (beautiful & responsive)  

--------------------------------------------
📌 Home Page:
--------------------------------------------
• Hero banner with stylish colors  
• "Featured Products" (use products from Fake Store API, limit 8)  
• Categories section  
• Sale/Offer banner  

--------------------------------------------
📌 Products Page:
--------------------------------------------
• Fetch products from: https://fakestoreapi.com/products  
• Grid layout  
• Filters (category – price – rating)  
• Product skeleton loading  

--------------------------------------------
📌 Product Details Page:
--------------------------------------------

• Large image, title, category, description  
• Price section  
• Add to Cart / Add to Wishlist  
• Quantity selector  

--------------------------------------------
📌 Cart Page:
--------------------------------------------
• List cart items (image – name – price – quantity – remove)  
• Subtotal & Total  
• "Proceed to Checkout" button  

--------------------------------------------
📌 Checkout Page:
--------------------------------------------
• Form: name, email, address, phone  
• Payment options section  
• Order Summary right side

--------------------------------------------
📌 About Page:
--------------------------------------------
• Modern layout  
• Mission + images + icons  

--------------------------------------------
📌 Contact Page:
--------------------------------------------
• Contact form  
• Map placeholder  
• Contact info

--------------------------------------------

  #4A60FF (primary)  
  #6C34FF (accent)  
  White + Gray backgrounds

--------------------------------------------
