# E-Commerce Project

## Project Overview
This is a full E-commerce website built with **React**, **Tailwind CSS**, and other modern frontend technologies. The project simulates a real-world online store, featuring dynamic product handling, a shopping cart, and a checkout process.

**Live Demo:** (https://stellar-commerce.netlify.app/)

---

## Technologies Used
- Frontend: React, Tailwind CSS, HTML, CSS, JavaScript
- State Management: React hooks, localStorage
- Styling: Tailwind CSS, responsive design
- Deployment: Netlify

---

## Features
- Fully responsive design (mobile, tablet, desktop)
- Home page with navbar, hero section, featured products
- Products page with grid layout, product cards, and dynamic add-to-cart
- Product details page
- Shopping cart with editable quantities, remove items, dynamic total
- Checkout page with a customer form (no real payment)
- About Us and Contact Us pages
- Smooth interactions and transitions
